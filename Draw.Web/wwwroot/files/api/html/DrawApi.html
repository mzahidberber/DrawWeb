CAD programlarında kullanmak üzere tasarlanmış bir hizmettir.Her kullanıcı kendi çeşitli çizim elemenlarını çizgi,çember,arc vb. oluşturup bunların verilerini kaydedebilir bu çizim elemanları üzerinden düzenlemeler yapabilir.Kullanıcılar oluşturdukları çizimleri .df formatına dönüştürebilirler.Bu hizmeti kullanabilmek için öncelikle auth hizmetinden yeni bir kullanıcı ve token oluşturmalı ve aşağıdaki isteklerin header kısmına bu token bilgisini ekleyip istek göndermelisiniz. Kullanıcı bilgilerini tokendan almaktadır.Projeye githup adresinden ulaşabilirsiniz.
