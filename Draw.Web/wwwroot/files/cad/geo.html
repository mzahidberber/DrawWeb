Geo hizmeti python ile geometrik hesaplamalar yapabilmek için tasarlanmış django api projesidir. Hizmette herhangi bir kayıt işlemi yapılmamaktadır.Sadece gönderilen isteğe göre hesaplamaları yapar ve geriye sonucu döner.Hizmeti kullanmak için token bilgisine gerek yoktur.Kullanmak için geo adresini ziyaret edebilirsiniz.
